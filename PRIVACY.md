# Codex Status Light Privacy Policy

Codex Status Light uses local, read-only access to inspect Codex session state on the user’s Mac.

## Data collection

- The app does not collect, upload, or share user data.
- The app does not use analytics, advertising, or tracking.
- The app reads local Codex state only to show the current status light.

## Local access

The app may request read-only access to the local `~/.codex` folder or related state files if automatic detection fails.
