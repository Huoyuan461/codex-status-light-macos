Codex Status Light Support

Codex Status Light is a lightweight, read-only macOS utility that shows the live status of your most recent local Codex session.

Getting Started

  • On first launch, the app tries to auto-detect your local ~/.codex folder.
  • If auto-detection fails, choose the .codex folder manually and grant read-only access.
  • Pick a display mode: notch, floating desktop, or menu bar only.
  • Red means disconnected, yellow means running, and green means completed or ready for your next prompt.

Troubleshooting

  • If no status appears, make sure Codex is running, then reopen the app or reauthorize the .codex folder.
  • If you do not see .codex, press Command + Shift + . in the file picker to reveal hidden files.
  • If the floating light is in the wrong place, switch to desktop mode and drag it anywhere you like.
