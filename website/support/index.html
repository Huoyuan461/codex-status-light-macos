Codex Status Light 支持

Codex Status Light 是一个只读的本机状态监控工具，用于显示最近活跃的 Codex 会话状态。

使用方法

  • 首次启动后选择本机的 .codex 文件夹并授予只读权限。
  • 选择“刘海旁”“桌面任意位置”或“仅菜单栏”。
  • 红灯表示异常中断，黄灯表示正在运行，绿灯表示任务已完成。

常见问题

  • 如果没有显示状态，请重新打开应用并重新授权 .codex 文件夹。
  • 如果找不到 .codex，请在文件选择窗口按 Command + Shift + . 显示隐藏文件。
  • 如果悬浮灯位置不合适，可以切换到桌面模式后直接拖动。
