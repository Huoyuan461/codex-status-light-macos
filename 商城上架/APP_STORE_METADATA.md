# App Store 元数据草案

- 应用名称：Codex Status Light
- 副标题：Codex 开发进度红绿灯
- 分类：开发者工具
- 价格：免费
- 最低系统：macOS 14.0

## 简短描述

通过菜单栏、刘海旁或桌面悬浮红绿灯，随时查看本机 Codex 任务状态。

## 完整描述

Codex Status Light 是一款轻量级 macOS 状态监控工具。它以扁平化红黄绿灯显示最近活跃的本机 Codex 会话：黄灯表示正在运行，绿灯表示已经完成，红灯表示任务异常中断。

应用支持菜单栏、刘海旁和桌面任意位置三种展示方式。所有状态分析均在本机完成，不需要 API Key，不上传会话内容，也不包含广告或跟踪功能。

## 关键词

Codex,开发工具,状态监控,菜单栏,效率,编程

## 审核备注

应用会在首次启动时优先自动识别本机 `~/.codex` 文件夹，只有在自动识别失败时才会请求用户手动选择。该授权仅用于只读分析本机 Codex 会话 JSONL 和 SQLite 状态文件。审核时如未安装 Codex，可以验证应用的自动识别、显示位置和空闲状态；完整状态变化需要本机 Codex 产生会话数据。
